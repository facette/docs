Bulk

Bulk requests execution

POST /api/v1/bulk/

This endpoint expects a request providing as body a list of API requests to execute in bulk, and returns a list of API responses corresponding to the requests. The format for describing an API request in a bulk list is:

{
  "endpoint": "<API endpoint relative to `/api/v1/` prefix>",
  "method": "<HTTP Method>",
  "params": {
    <query string parameters>
  }
}
