Server API

Get Items Statistics

GET /api/v1/stats

Returns a statistics object along with back-end items information.

Response:

{
    "origins": 1,
    "sources": 3,
    "metrics": 353,
    "graphs": 1,
    "collections": 1,
    "sourcegroups": 1,
    "metricgroups": 4
}
